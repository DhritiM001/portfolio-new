Lending Club Case Study

Project Summary

Goal: Help Jasmine, a peer-to-peer investor, make smarter lending decisions using data analytics.

Role: Business Analyst - modeling borrower risk, designing investment strategies, and optimizing portfolio returns.

Tools Used: Python, PCA, K-Means Clustering, Logistic Regression, Lasso/Ridge, Excel Solver


Project Timeline


Problem Framing: Can We Invest Smarter?

The Challenge: Lending Club assigns loan grades (A–G) to borrowers, but are these reliable indicators of risk?

Our Hypothesis: By applying machine learning, we can uncover borrower patterns, improve risk prediction, and enhance returns.


Step 1: Diagnosing Patterns with Descriptive Analysis

Techniques Used:

1. Principal Component Analysis (PCA) to reduce feature complexity

2. K-Means Clustering (k=4) to segment borrower profiles


Key Insight: Loan grades were correlated with cluster risk levels, but not perfectly. For example, some Grade C loans were safer than Grade B. We visualized this with cluster profiles:

1. Cluster 0: Low-risk, small loans, aligned with Grades A–B

2. Cluster 3: High-risk, subprime borrowers, mostly Grades E–G


Step 2: Predicting Outcomes with Machine Learning

Approach: Built 2 models -

1. Logistic Regression to predict loan defaults

2. Lasso Regression to predict investor returns


Results:

1. Logistic model achieved ~65% accuracy in identifying default risks

2. Lasso model predicted returns with less than 3% absolute error on average


Decision trees explained loan risk with easy-to-follow rules (e.g., "High loan + Low income = High default risk").


Step 3: Designing Real-World Investment Strategies

We created 7 investment strategies to help Jasmine invest smarter:


Hybrid strategy stood out as a strong all-rounder.


Step 4: Optimizing the Portfolio with Constraints

Tool: Linear Optimization

Goal: Maximize return while staying within Jasmine’s budget


Approach:

1. Formulated an optimization problem

2. Applied constraints (budget, loan caps, risk flags)

3. Solver selected the most promising loan mix


Results:

1. Optimized portfolio return: 18–21%, up from 8% in random strategy

2. Strategy scaled well across budgets, balancing growth and risk


Final Recommendation

We recommend Jasmine use the optimized portfolio strategy if tools like Gurobi/Excel Solver are available. If not, the Hybrid Strategy offers a reliable, scalable alternative with high returns and lower risk.


Considerations & Assumptions

1. Data Limitation: Based on 2014–15 Lending Club data; borrower behavior may evolve

2. Assumptions: Simplified reinvestment and return formulas

3. Risk: Cluster-based risk scores depend on model accuracy


Key Takeaways

Business Value: Grew investor return potential from 8% to 18–21%

Analytical Depth: Combined unsupervised learning, predictive modeling, and prescriptive analytics

Scalability: Designed solutions that adapt across investment sizes and market shifts
